Downloading Praat for FreeBSD

The FreeBSD edition of Praat is maintained by Jason Bacon and Adriaan de Groot and can be downloaded from its FreeBSD ports download page.

If you prefer (e.g. if you want a newer version), you could download the Praat source code yourself and compile it.
